# 脉冲激光控制模块

> STM32F103C8T6 实时脉冲选通器 — 从连续脉冲串中选取 N 个脉冲输出，组间延时可调。

---

## 1. 概述

本模块接收外部 **1KHz ~ 100KHz TTL 脉冲信号**，通过实时门控逻辑，按设定数量（N）选通每组脉冲，并在组间插入可调的静音延时（T）。参数通过串口设置，掉电自动保存。

### 核心功能

1. **脉冲可选（N）**：从连续脉冲串中选取 1 ~ 65535 个脉冲输出
2. **延时可调（T）**：每组脉冲之间插入 1 ~ 50000 ms 低电平静音
3. **实时跟随**：输出波形完整复制输入脉宽，输入→输出延迟 ~2μs

## 6. 串口指令

波特率 **115200**, 8 数据位, 1 停止位, 无校验, 无流控。命令以 `\r` 或 `\n` 结尾。

| 命令 | 功能 | 示例 |
|------|------|------|
| `N=<1~Nmax>` | 设置脉冲数 + 保存，Nmax 随输入频率动态变化 | `N=5` |
| `T=<1~50000>` | 设置延时(ms) + 保存 | `T=10` |
| `N=<n> T=<t>` | 同时设置两个参数 + 保存 | `N=5 T=10` |
| `STATUS?` | 查询当前参数、输入频率、N 上限 | `STATUS?` |

### 命令响应示例

```
→ N=5 T=1
← OK N=5 T=1ms

→ STATUS?
← Fin=100000Hz N=5/100000 T=1ms     ← 100KHz 输入时 N 最大 100000

→ T=50
← OK N=5 T=50ms

→ N=200000                           ← 超过当前频率允许的 Nmax
← ERR N RANGE (1~100000)             ← 上限随输入频率动态变化

→ T=0
← ERR T RANGE (1~50000)
```

## 8. 技术实现

### 架构

```
PA0 (输入脉冲) ──→ TIM2 双沿捕获 ──→ 实时门控状态机 ──→ PA1 (输出)
                       │                    │
                  13.9ns 分辨率        S_FORWARD: 跟随
                                      S_SILENCE: 静音 (TIM3 1ms 计时)
```

- **TIM2 ISR**：最高优先级 (0,0)，捕获 PA0 边沿，直接控制 PA1 翻转，延迟 ~2μs
- **TIM3 ISR**：优先级 (1,0)，每 1ms 溢出一次，递减静音计数器
- **USART1 ISR**：优先级 (2,0)，接收串口命令

### 存储

- 地址：Flash 第 63 页 `0x0800FC00`
- 内容：`burst_count` (4B) + `burst_delay` (4B)
- 首次上电：擦除态 `0xFFFFFFFF` → 使用默认值 N=2, T=2
- 触发保存：每次 `N=` 或 `T=` 命令
