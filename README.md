# 脉冲激光控制模块

## 1. 概述

本模块用于控制磁光开关（通过 TTL 高低电平），从连续激光脉冲中选通指定数量的脉冲输出。支持**透传模式**和**固定延时模式**，参数可通过串口命令设置并掉电保存。

### 硬件平台

| 项目 | 规格 |
|------|------|
| MCU | STM32F103C8T6（Cortex-M3, 72MHz, 64KB Flash, 20KB RAM） |
| 串口芯片 | CH340N（USB-UART，板载 Type-C） |
| 输入防抖 | BNC → PA0（10K 下拉电阻） |
| 输出驱动 | PA1 → 接线端子（TTL 3.3V） |
| 烧录接口 | SWD（PA13-SWDIO, PA14-SWCLK） |

## 2. 引脚定义

| 引脚 | 方向 | 功能 |
|------|:--:|------|
| **PA0** | 输入 | 脉冲信号输入（TIM2 CH1 捕获，3.3V TTL） |

| **PA9** | TX | 串口发送 → CH340N RXD |
| **PA10** | RX | 串口接收 → CH340N TXD |
| **PA13** | SWDIO | 程序烧录 |
| **PA14** | SWCLK | 程序烧录 |

## 3. 参数规格

### 3.1 输入信号

| 参数 | 范围 | 说明 |
|------|------|------|
| 电平 | 0 ~ 3.3V TTL | 高电平有效 |
| 重复频率 | **1 kHz ~ 100 kHz** | 周期 10μs ~ 1ms |
| 最小高电平脉宽 | **~1.4μs** | TIM2 在期间捕获边沿，需要两个沿至少隔 1.4μs |
| 脉冲数量 | 1 ~ N（不限） | 连续或间歇 |
| 输入阻抗 | ~10KΩ 下拉 | 防浮动噪声 |

### 3.2 输出信号

| 参数 | 范围 | 说明 |
|------|------|------|
| 电平 | 0 ~ 3.3V TTL | Push-Pull，50MHz 驱动 |
| 透传模式延迟 | **~3μs** | TIM2 ISR + GPIO 响应 |
| 固定延时模式延迟 | **≤1 脉宽** | 必须先测量宽度才能复制、替换低电平 |
| 固定低电平时长 | **1 ~ 50000 ms** | 1ms 步进，TIM3 1kHz 计时 |
| 输出阻抗 | 100Ω 串联保护 | 防外部短路 |

### 3.3 串口通信

| 参数 | 值 |
|------|------|
| 波特率 | **115200** |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | 无 |
| 流控 | 无 |
| 换行符 | `\r` 或 `\n`（回车或换行均可） |

## 4. 工作模式

模块通过 `gap_ms` 参数自动切换工作模式：

### 4.1 透传模式（`GAP=0`，默认）

- 输入波形完整复制到输出
- 高电平宽度和低电平间隔完全保留
- 延迟 **~3μs**（TIM2 中断响应 + GPIO 写入）
- 适用场景：需要原样输出输入波形

```
输入: ┌──┐  ┌──┐  ┌──┐
      └──┘  └──┘  └──┘

输出: ┌──┐  ┌──┐  ┌──┐
      └──┘  └──┘  └──┘       (完全镜像)
```

### 4.2 固定延时模式（`GAP=1` ~ `50000`）

- 高电平宽度复制输入
- **低电平间隔替换为串口设置的固定值**
- 延迟：**≤1 脉宽**（需先测量高电平宽度）
- 适用场景：需要统一隔开脉冲间距

```
输入: ┌──┐  ┌──┐  ┌──┐
      └──┘  └──┘  └──┘
       500   500   500  (μs)

输出: ┌──┐    ┌──┐    ┌──┐
      └──┘    └──┘    └──┘
       500 ↑1ms↑ 500  ↑1ms↑   (GAP=1)
```

## 5. 串口命令集

所有命令大小写不敏感，以 `\r` 或 `\n` 结尾。

| 命令 | 参数 | 功能 | 示例响应 |
|------|------|------|----------|
| `GAP=0` | — | 切换到**透传模式**，自动保存 | `OK MODE=PASSTHROUGH` |
| `GAP=<ms>` | 1~50000 | 设置固定低电平时长(ms)，自动保存 | `OK GAP=1ms` |
| `GAP?` | — | 查询当前 GAP 值/模式 | `GAP=1ms` 或 `MODE=PASSTHROUGH` |
| `MODE?` | — | 查询当前工作模式 | `MODE=PASSTHROUGH` 或 `MODE=GAP GAP=1ms` |
| `SAVE` | — | 手动保存当前参数到 Flash | `SAVED` 或 `SAVED PASSTHROUGH` |
| `STATUS?` | — | 查询 FIFO 使用量 + 模式 | `FIFO=0 MODE=PASSTHROUGH` |
| `HELP` | — | 显示帮助 | `GAP=<ms>(0~50000) GAP=0=passthrough GAP? MODE? STATUS? SAVE HELP` |

> **注意：** `GAP=` 设置后自动保存到内部 Flash（最后 1KB 页），掉电不丢失。首次烧录默认 `GAP=0`（透传模式）。

## 6. 上位机操作流程

### 6.1 环境准备

使用任意串口调试助手（如 SSCOM、PuTTY、Tera Term 等），配置：

```
波特率: 115200
数据位: 8
停止位: 1
校验位: None
流控:   None
```

### 6.2 基本操作示例

**示例 1：透传模式（默认）**

```
上电 → 串口收到: LASER-CTRL
串口收到: LOAD GAP=0ms（或空，首次烧录无输出）
此时模块处于透传模式，输入即输出，无需任何操作。
```

**示例 2：切换到 1ms 固定低电平**

```
发送: GAP=1
收到: OK GAP=1ms
此时输入脉冲的高电平被复制，低电平固定为 1ms。
```

**示例 3：查询状态**

```
发送: STATUS?
收到: FIFO=3 MODE=GAP GAP=1ms
表示 FIFO 中还有 3 个条目待输出，当前为 GAP 模式，低电平 1ms。
```

**示例 4：切回透传**

```
发送: GAP=0
收到: OK MODE=PASSTHROUGH
立即回到透传模式。
```

**示例 5：掉电恢复**

```
1. 发送: GAP=5
2. 收到: OK GAP=5ms
3. 断开电源，重新上电
4. 收到: LASER-CTRL
         LOAD GAP=5ms
5. 模块自动恢复到 GAP=5ms 模式，无需重新设置。
```

### 6.3 常用配置速查

| 场景 | 命令 | 效果 |
|------|------|------|
| 恢复默认透传 | `GAP=0` | 输入直通输出 |
| 脉冲间隔 1ms | `GAP=1` | 脉冲间 1ms 低电平 |
| 脉冲间隔 50ms | `GAP=50` | 脉冲间 50ms 低电平 |
| 脉冲间隔 1s | `GAP=1000` | 脉冲间 1000ms 低电平 |
| 脉冲间隔 50s | `GAP=50000` | 脉冲间 50000ms 低电平（上限） |
| 查看当前模式 | `MODE?` | 返回模式/延时 |
| 手动保存 | `SAVE` | 写入 Flash |
| 查看状态 | `STATUS?` | FIFO + 模式 |

## 7. 技术实现细节

### 7.1 数据流

```
PA0 (输入)
  │
  ▼
TIM2 CH1 双沿捕获 (72MHz, 13.9ns 分辨率)
  │
  ├─ 上升沿 → 记录时间戳
  │
  └─ 下降沿 → 计算高电平脉宽 (now - start) / 72 = μs
                    │
                    ▼
             FIFO (1024 条目, bit15 标记宽/间隙)
                    │
                    ▼
             Output_Poll() 主循环分发：
                    │
         ┌──────────┴──────────┐
         ▼                     ▼
    gap==0 透传           gap>0 固定延时
         │                     │
         ▼                     ▼
   TIM4(脉宽)            TIM4(脉宽)
   高电平                高电平
         │                     │
         ▼                     ▼
   FIFO 取 gap           TIM3(gap_ms)
   TIM4(gap)             低电平
   低电平
```

### 7.2 定时器分配

| 定时器 | 时钟 | 用途 |
|------|------|------|
| **TIM2** | 72MHz, /1, 16bit | 输入脉冲双沿捕获，13.9ns 分辨率，最高优先级 (0,0) |
| **TIM4** | 1MHz（/72）, 16bit | 输出脉宽计时，1μs 精度，最大 65.5ms，优先级 (1,0) |
| **TIM3** | 1kHz（/72000）, 16bit | 固定低电平计时，1ms 精度，最大 65.5s，优先级 (1,1) |

### 7.3 中断优先级

| 中断 | 抢占优先级 | 子优先级 | 说明 |
|------|:--:|:--:|------|
| TIM2 | **0** | 0 | 捕获中断，最高优先级，确保测量精度 |
| TIM4 | 1 | 0 | 输出脉宽到期 |
| TIM3 | 1 | 1 | 输出低电平到期 |
| USART1 | 2 | 0 | 串口接收，最低优先级 |

### 7.4 Flash 存储

- 存储地址：**0x0800FC00**（第 63 页，最后 1KB）
- 存储内容：`gap_ms`（4 字节）
- 首次上电：Flash 擦除态 0xFFFFFFFF → 判断为未配置 → 保持 `gap_ms=0`（透传）
- 写入次数：每次 `GAP=` 或 `SAVE` 触发；Flash 寿命 ≥10,000 次擦写
