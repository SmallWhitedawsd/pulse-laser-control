# 脉冲激光控制模块

基于 STM32F103C8T6 的实时脉冲选通器。接收外部 1KHz~100KHz 脉冲信号，按设定数量输出每组脉冲，每组之间延时可调。

## 硬件

| 项目 | 参数 |
|------|------|
| MCU | STM32F103C8T6 (Cortex-M3, 72MHz) |
| 输入引脚 | PA0 (3.3V TTL, 内部下拉) |
| 输出引脚 | PA1 (3.3V TTL, 推挽 50MHz) |
| 串口 | USART1, PA9(TX)/PA10(RX), 115200-8N1 |
| USB 芯片 | CH340N (Type-C) |
| 调试 | SWD (PA13/PA14) |

## 参数

| 参数 | 缩写 | 范围 | 默认值 | 说明 |
|------|------|------|--------|------|
| 脉冲数 | N | 1 ~ 65535 | 2 | 每组输出的高电平脉冲数 |
| 延时 | T | 1 ~ 50000 ms | 2 ms | 每组脉冲之间的静音时间 |
| 输入频率 | — | 1KHz ~ 100KHz | — | 外部信号发生器提供 |

## 串口指令

```
N=5 T=1    设置脉冲数 5、延时 1ms，自动保存到 Flash
STATUS?    查询当前 N、T 值
```

### 示例

```
发送: N=5 T=1
回复: OK N=5 T=1ms

发送: STATUS?
回复: N=5 T=1ms
```

> 每条指令发完后等收到 OK 再发下一条。Flash 写入约需 20ms，期间不响应串口。

## 工作原理

```
外部信号 → PA0(TIM2 双沿捕获) → 实时门控 → PA1 输出
                                   │
                          ┌────────┘
                          │ 计 N 个上升沿
                          │ 第 N 个下降沿后关门
                          │ 延时 T ms 后开门
```

TIM2 捕获 PA0 上升/下降沿，ISR 内直接控制 PA1 电平。门开时 PA1 实时跟随 PA0（延迟 ~2μs），门关时 PA1 保持低电平。

## 波形示意

输入 1KHz 50% 占空比，N=2，T=1ms：

```
输入:  _|‾|_|‾|_|‾|_|‾|_|‾|_|‾|_
输出:  _|‾|_|‾|___|‾|_|‾|___|‾|_|‾|___
        p1  p2 静音  p1  p2 静音
```
